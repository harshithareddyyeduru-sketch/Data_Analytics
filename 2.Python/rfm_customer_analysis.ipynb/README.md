Retail Sales Analysis - Python Layer

This folder contains Python scripts used for advanced analytics.

Files Included:

data_loader.py Connects Python to the PostgreSQL database.

rfm_segmentation.py Calculates Recency, Frequency, and Monetary scores for customers.

market_basket_analysis.py Performs association rule mining to identify products that are frequently purchased together.

rfm_customer_analysis.ipynb Exploratory notebook for customer behavior analysis.
