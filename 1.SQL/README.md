Retail Sales Analysis - SQL Layer

This folder contains SQL scripts used for the data preparation and analysis stages of the Retail Analytics project.

SQL Pipeline:

Database Setup Creates tables and loads raw data.

Data Cleaning Handles missing values and incorrect formats.

Star Schema Creates fact and dimension tables for analytics.

RFM Analysis Calculates Recency, Frequency, Monetary scores.

Cohort Analysis Analyzes customer retention patterns.

Business Queries Generates insights used in the Power BI dashboard.
